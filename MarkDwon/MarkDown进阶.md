# MarkDown的一些~~有趣~~进阶玩法

## 目录

[toc]

只需要`[toc]`就可以自动生成一个目录，并且带有超链接，可以链接到当前文件的所有标题上，快速跳转

## 删除线
单词前后加两个波浪号即可
适用于~~滑稽~~删除垃圾信息~~并希望垃圾信息被看到，哈哈哈~~

---
## 任务列表语法
任务项前加短横线，空格，再加[x]就行啦
我们来试一下
***方括号中加x表示打上钩，空格表示不打钩***
- [x] 吃饭
- [ ] 睡觉
- [ ] 打豆豆
    - [ ] 哈哈哈
    - [ ] 甚至还支持嵌套

---
## 表格
- 三个或多个连字符"-"来创建每列的标题，并用"|"分割每列
    | 姓名 | 班级 |
    | ----- |---|
    |王德发|寄21-2|
- 连字符的左侧、右侧、两边加冒号可以实现左对齐、右对齐、居中
    | 姓名 | 班级 |
    | :------: |:----:|
    |王德发|寄21-2|
