# 插入

直接无脑`i`就行了，不记别的啦

# 移动光标

- `M`：移动到中间行
- `L`：移动到最后一行行首
- `gg`：移动到文件开头
- `G`：移动到文件末尾

# 删除

- `dd`：删除光标所在行，`n dd`删除n行
- `D`：删除光标后所有内容，包括光标
- `d0`：删除光标前所有内容

# 撤销

- `u`：撤销一步
- `U`：全部撤销
- `Ctrl-r`：反撤销

# 重复命令

- `.`：重复上一次操作的命令

# 赋值粘贴

- `yy`：赋值当前行，`n yy`赋值n行
- `p`：在光标所在位置向下新开辟一行粘贴

# 查找

`/ 字符串`，查找字符串

- `n`：下一个
- `N`：上一个

# 分屏

**注意：要在冒号中使用**

- `sp`：上下分屏，后可跟文件名
- `vsp`：左右分屏，后可跟文件名
- `Ctrl+w+w`：多个窗口切换
- `Ctrl+w c`：关闭当前窗口
- `Ctrl+w q`：关闭当前窗口，若是最后一个，则退出vim
