# 单例模式

## 特点

1. 类最多只能有一个实例
2. 能自主创建这个唯一的实例（非调用者创建）
3. 能够向全局提供公共的访问点来获取唯一的实例

```c++
#include <iostream>

using namespace std;

class CSingleton
{
private:
	static CSingleton* ssr;

private:
	CSingleton() : m_a(1) {} // 构造函数私有化
	//CSingleton(const CSingleton&) {} // 拷贝构造私有化
	CSingleton(const CSingleton&) = delete; // 或者也可以弃用拷贝构造

public:
	int m_a;

public:
	static CSingleton* GetSingleton()
	{
		if (!ssr) {
			ssr = new CSingleton;
		}
		return ssr;
	}

	static void DestorySingleton(CSingleton*& psin)
	{
		if (ssr) {
			delete ssr;
			ssr = nullptr;
		}
		psin = nullptr;
	}
};

// 静态的指针在类外初始化
CSingleton* CSingleton::ssr = nullptr;

int main()
{
	CSingleton* p = CSingleton::GetSingleton();
	CSingleton* p2 = CSingleton::GetSingleton();

	//CSingleton sin(*p); // 拷贝构造应当被禁止

	cout << p << '\t' << p2 << endl;
	p->m_a = 666;
	cout << p2->m_a << endl;

	CSingleton::DestorySingleton(p);
	if (p) {
		cout << p->m_a << endl;
	}
	else {
		cout << "单例不可用" << endl;
	}

	p = CSingleton::GetSingleton();
	if (p) {
		cout << p->m_a << endl;
	}
	else {
		cout << "单例不可用" << endl;
	}

	return 0;
}
```

## 懒汉式

在调用获取的接口时去创建唯一的实例，**时间换空间的做法**

```c++
#include <iostream>

using namespace std;

class CSingleton
{
private:
	static CSingleton* ssr;

private:
	CSingleton() : m_a(1) {} // 构造函数私有化
	//CSingleton(const CSingleton&) {} // 拷贝构造私有化
	CSingleton(const CSingleton&) = delete; // 或者也可以弃用拷贝构造

public:
	int m_a;

public:
	static CSingleton* GetSingleton()
	{
		// 加锁
		if (!ssr) {
			ssr = new CSingleton;
		}
		// 解锁
		return ssr;
	}

	static void DestorySingleton(CSingleton*& psin)
	{
		if (ssr) {
			delete ssr;
			ssr = nullptr;
		}
		psin = nullptr;
	}
};

// 静态的指针在类外初始化
CSingleton* CSingleton::ssr = nullptr;

int main()
{
	CSingleton* p = CSingleton::GetSingleton();
	CSingleton* p2 = CSingleton::GetSingleton();

	//CSingleton sin(*p); // 拷贝构造应当被禁止

	cout << p << '\t' << p2 << endl;
	p->m_a = 666;
	cout << p2->m_a << endl;

	CSingleton::DestorySingleton(p);
	if (p) {
		cout << p->m_a << endl;
	}
	else {
		cout << "单例不可用" << endl;
	}

	p = CSingleton::GetSingleton();
	if (p) {
		cout << p->m_a << endl;
	}
	else {
		cout << "单例不可用" << endl;
	}

	return 0;
}
```

### 优化

```c++
#include <iostream>

using namespace std;

class CSingleton
{
private:
	CSingleton() : m_a(1) {} // 构造函数私有化
	//CSingleton(const CSingleton&) {} // 拷贝构造私有化
	CSingleton(const CSingleton&) = delete; // 或者也可以弃用拷贝构造

public:
	int m_a;

public:
	static CSingleton* GetSingleton() // 多线程下，不会创建出多个实例
	{
		static CSingleton ssr; // 第一次调用这个函数，创建实例才会存在
		return &ssr;
	}
};

int main()
{
	CSingleton* p = CSingleton::GetSingleton();
	CSingleton* p2 = CSingleton::GetSingleton();

	cout << p << '\t' << p2 << endl;

	p->m_a = 20;
	cout << p2->m_a << endl;

	return 0;
}
```

## 饿汉式

在程序创建之初，唯一的实例就会被创建，**空间换时间的做法**

**优点：多线程下，不会创建多个实例**

而且由于是静态成员，是储存在全局/静态区的，不需要手动释放内存

```c++
#include <iostream>

using namespace std;

class CSingleton
{
private:
	static CSingleton ssr; // 编译器存在，只存在一份

private:
	CSingleton() : m_a(1) {} // 构造函数私有化
	//CSingleton(const CSingleton&) {} // 拷贝构造私有化
	CSingleton(const CSingleton&) = delete; // 或者也可以弃用拷贝构造

public:
	int m_a;

public:
	static CSingleton* GetSingleton() // 多线程下，不会创建出多个实例
	{
		return &ssr;
	}
};

CSingleton CSingleton::ssr; // 唯一的实例，定义初始化

int main()
{
	CSingleton* p = CSingleton::GetSingleton();
	CSingleton* p2 = CSingleton::GetSingleton();

	cout << p << '\t' << p2 << endl;

	p->m_a = 20;
	cout << p2->m_a << endl;

	return 0;
}
```
